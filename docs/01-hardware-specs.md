# Geekom AX8 Hardware Specifications

## Overview

![Geekom AX8 Mini PC](https://www.geekompc.com/wp-content/uploads/2024/07/AX8-PRO-PWOER.webp)

The Geekom AX8 is a powerful mini PC suitable for software development, gaming, and multimedia applications.

## Specifications

### Processor
- AMD Ryzen 9 8945HS (Hawk Point)
- 8 Cores / 16 Threads
- Base Clock: 4.0 GHz
- Boost Clock: Up to 5.2 GHz
- TDP: 35-54W configurable

### AI Capabilities
- **Ryzen AI (XDNA NPU)**: Dedicated neural processing unit
- **AI Performance**: Up to 16 TOPS (Tera Operations Per Second)
- **AI Engine**: AMD XDNA architecture for efficient AI workloads
- **Windows Studio Effects**: AI-enhanced video conferencing (Windows)
- **Linux AI Support**: Compatible with ONNX Runtime, OpenVINO, and DirectML
- **Use Cases**: 
  - AI-accelerated image/video processing
  - Real-time background blur and noise suppression
  - Local AI model inference (LLMs, image generation)
  - Machine learning development and testing

### Graphics
- AMD Radeon 780M integrated GPU
- 12 RDNA 3 Compute Units
- Up to 2.8 GHz GPU Clock
- Supports 4K displays

### Memory
- Up to 64GB DDR5-5600 MHz
- Dual-channel configuration
- SO-DIMM slots (expandable)
- **AMD EXPO Support**: Extended Profiles for Overclocking
  - EXPO is AMD's equivalent to Intel XMP for memory overclocking
  - Enables one-click memory performance optimization
  - Compatible with DDR5 EXPO-certified memory modules
  - Automatically configures optimal timings and voltages
  - Can achieve speeds beyond JEDEC standards (up to DDR5-6400+ with compatible modules)
  - Check BIOS settings to enable EXPO profiles
  - Recommended for users wanting maximum memory performance

### Storage
- M.2 2280 NVMe SSD (PCIe 4.0)
- Support for up to 2TB primary drive
- Additional M.2 2242 slot available

### Connectivity
- 2x USB4 ports (40 Gbps)
- 4x USB 3.2 Gen2 Type-A ports
- 2x HDMI 2.1 ports
- 1x DisplayPort 1.4
- 2.5G Ethernet (RJ45)
- Wi-Fi 6E (802.11ax)
- Bluetooth 5.2

### Audio
- 3.5mm audio jack
- Built-in microphone
- Digital audio out via HDMI/DisplayPort

### Dimensions
- Approx. 130 x 130 x 50mm
- Compact form factor suitable for desk or entertainment center

## Performance Expectations

### Software Development
- Excellent for compiling large codebases
- Multiple virtual machines/containers supported
- Ideal for full-stack development

### Gaming
- Capable of 1080p gaming on medium-high settings
- Good for indie games and esports titles
- Some AAA games at lower settings

### Video Streaming
- Hardware-accelerated encoding/decoding
- 4K video playback and streaming
- Multiple display support

### Virtualization
- Hardware virtualization support (AMD-V)
- IOMMU support for PCIe passthrough
- Suitable for running multiple VMs

### AI & Machine Learning
- NPU acceleration for AI workloads
- Efficient inference for small to medium AI models
- Local AI processing without cloud dependencies
- Suitable for AI development and testing
- Compatible with popular ML frameworks (PyTorch, TensorFlow, ONNX)
